## Información que recopilamos

Spiral Word Puzzle recopila una cantidad limitada de información del usuario únicamente para la funcionalidad de la tabla de clasificación.

- **Información de autenticación (requerida):**  
  Al enviar una puntuación alta, recopilamos un nombre de usuario o identificador único para asociar las puntuaciones con los jugadores.

- **Ubicación del país (opcional):**  
  Puedes elegir opcionalmente tu país de una lista al enviar una puntuación alta. Esta información no se recopila automáticamente y solo se almacena si la seleccionas.

## Almacenamiento y uso de los datos

Toda la información recopilada se almacena de forma segura en una base de datos SQL privada alojada por el desarrollador de la aplicación.  
No se recopila información personal como dirección de correo electrónico, nombre real, número de teléfono o ubicación del dispositivo.

Los datos se utilizan únicamente para mostrar puntuaciones en la tabla de clasificación y nunca se venderán ni compartirán con terceros.

## Servicios de terceros

La aplicación utiliza los siguientes SDK y dependencias:

- **React Native Firebase (solo Core):**  
  Se utiliza para la funcionalidad de la aplicación, no para análisis ni informes de fallos.

- **AdMob (react-native-google-mobile-ads):**  
  Puede recopilar datos anónimos del dispositivo y de uso según las políticas de anuncios de Google:  
  https://policies.google.com/technologies/ads  
  Los usuarios menores de 13 años reciben anuncios apropiados para niños.

- **Otros módulos:**  
  SDK como AsyncStorage, Expo, Lottie y el selector de países se utilizan solo para funcionalidad local y no recopilan datos del usuario.

Si realizamos cambios en la aplicación que afecten esta política, actualizaremos esta página en consecuencia.
